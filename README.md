## About

I'm Joe Bylund — lead software engineer at [Klaviyo](https://www.klaviyo.com/), with a background in distributed systems, data pipelines, and search infrastructure. I did a PhD in computational biophysics at Columbia before moving into industry, where I've worked on systems moving tens of billions of events per day, sub-millisecond query engines, and everything in between.

Current focus: building [Arcane Tutor](https://github.com/jbylund/arcane_tutor), a self-hosted Magic: The Gathering card search engine with extended query syntax that Scryfall doesn't support.
